** Fake News Detection Using Machine Learning

## Project Overview
This project is a Machine Learning-based Fake News Detection System that classifies news articles as 'Real' or 'Fake'. 

## Features
- Detects whether a news article is Real or Fake.
- TF-IDF Vectorization for feature extraction.
- User can enter custom news articles for prediction.

## Technologies Used
- Python
- Google Colab
- Pandas
- NumPy
- Scikit-learn
- NLTK
- Matplotlib

## Machine Learning Models
- Logistic Regression
- Decision Tree Classifier
- Random Forest Classifier
- Gradient Boosting Classifier

## How to Run
1. Clone or download this repository.
2. Download the dataset from the Kaggle link above.
3. Open the notebook in Google Colab or Jupyter Notebook.
4. Upload the dataset files (`Fake.csv` and `True.csv`).
5. Run all notebook cells.
6. Enter a news article to check whether it is Real or Fake.
